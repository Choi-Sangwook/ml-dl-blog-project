## 게시 전 반드시 수정

없음.

NumPy softmax 데모는 최종본의 Python 코드 블록을 그대로 추출해 실행했다. 확률 값, 선택 표시, 선택 단어는 문서의 "실제 실행 출력"과 모두 일치한다.

검증 실행 출력:

```text
[1번째 위치] 확률 합 = 1.0000
         Je : 0.750  <== 선택
         Tu : 0.075
         Il : 0.046
     t'aime : 0.041
       aime : 0.050
      <end> : 0.037
  -> 선택된 단어: Je
[2번째 위치] 확률 합 = 1.0000
         Je : 0.034
         Tu : 0.031
         Il : 0.038
     t'aime : 0.765  <== 선택
       aime : 0.085
      <end> : 0.047
  -> 선택된 단어: t'aime
```

최종본의 출력 블록에는 두 step 사이에 가독성용 빈 줄이 1줄 들어 있다. 값 차이는 아니므로 게시 전 필수 수정으로 보지는 않는다.

## 선택적으로 개선

- NumPy 출력 블록을 실제 stdout과 문자 단위로 완전히 맞추려면, 최종본의 출력 블록에서 두 step 사이의 빈 줄을 제거하거나 Python 코드의 루프 마지막에 `print()`를 추가하면 된다.
- §8 요약의 `GNMT(2016)는 LSTM enc-dec + attention -> 이후 Transformer가 attention 중심`은 🟩로 표시되어 있어 출처 구분은 맞다. 더 엄밀히 쓰려면 "Transformer 계열에서 attention이 더 중심적인 구조로 쓰임"처럼 문장형으로 풀어도 좋다.
